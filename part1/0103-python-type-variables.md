# 타입(Type)

### String

- 문자, 단어 등으로 구성된 문자들의 집합   

- Python에서는 작은('')/큰 따옴표("")로 둘러쌓여 있으면 문자열로 인식
	-  'Python'
	-  '1234'

### Integer

-	수학 : 자연수(1, 2, 3, ...) 와 이들의 음수(-1, -2, -3, ...)와 0으로 이루어진 수 체계   
-  컴퓨터 과학 : 소숫점으로 표현해야되는 숫자가 없는 수

	- 123
	- 10

### Float

-	Floating-point numbers 부동 소숫점 숫자
-  부동 소숫점 방식으로 표현해야할 값이 있는 숫자
	-  	1.234
	-  10.0






# 변수(Variables)

- 수학: 수식에 따라 변하는 값

`y = 2x + 1`

- 컴퓨터과학: 아직 알려지지 않았거나 어느정도 까지만 알려진 양 또는 정보

```python
animal = "cat"
print(animal)

animal = "dog"
print(animal)
```
